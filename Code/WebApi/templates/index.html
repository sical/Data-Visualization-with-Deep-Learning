Know Your Charts Test it

Description

This project, aim to help us see charts differently. The deep learning discriminative model allows us to put charts into categories. While classifying chart, a convolutional neuronal network, is forced to "compress" characteristics of the given picture as he have as many inputs as pixel but only one output. So we can see what makes ,for example , a bar chart seen as bar chart by this model. Through this web site, you will be able to "test" your charts to see if the neuronal network can find out what it is, and "compare" it to the norms. You can also see the limits of models and seek for fool the model with ambiguous pictures. More information at our github


Tensorflow
Tensorflow is a powerfull deep learning tool with some examples. Tensorflow can be considered as solution as a lot of operations are done in background. Even if tensorflow have some build in function, a lot of work have to be done manually. We used python as an API to make this model. Coding in tensorflow can be summarized as make graph nodes and feeding them with tensor flowing through the graph. Our is based on the google's model named inceptionv3 trainned with imagenet dataset (1000 classes and more than 1 million pictures). We used transfer learning to retrain this model to classify our 4 categories. This modelhave 97.5% succes rate Tensorflow home page

Nanonets
Nanonets is a user friendly online solution. It allows users to simply build a model from their own images. Nanonets also use transfer learning on inceptionV3 model. You just have to provide the pictures and "label" them (upload them into the right categories).This model have a 98.2% success rate
